# MyApp

Мобильное приложение на React Native + Expo с подпиской-гейтом.

## Архитектура

Expo Router (файловый роутинг, SDK 55) с односторонним флоу:
запуск → проверка AsyncStorage → онбординг → пейвол → домашний экран.
После покупки приложение всегда открывается сразу на /home.
Хранилище — AsyncStorage, никакого внешнего стейт-менеджера нет.

## Структура проекта

| Файл | Назначение |
|------|------------|
| `src/app/_layout.jsx` | Корневой лейаут, проверка подписки, спиннер |
| `src/app/index.jsx` | Заглушка на время редиректа |
| `src/app/onboarding.jsx` | Онбординг с кнопкой Continue |
| `src/app/paywall.jsx` | Выбор плана, эмуляция покупки |
| `src/app/home.jsx` | Основной экран с контентом |
| `src/storage/subscription.js` | AsyncStorage: saveSubscription / checkSubscription |

## Что улучшил бы при большем времени

- **Реальный IAP** — интеграция `expo-in-app-purchases` или RevenueCat вместо эмуляции
- **Онбординг** — несколько слайдов с пагинацией вместо одного экрана
- **Тесты** — покрытие хелперов AsyncStorage и флоу навигации через Jest
- **Локализация** — i18n с `expo-localization`
- **Тема** — поддержка тёмного режима через `useColorScheme`
